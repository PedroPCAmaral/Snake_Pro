🐍 Snake PRO – Jogo da Cobrinha Avançado

Projeto desenvolvido em HTML, CSS e JavaScript puro, com foco em lógica de programação, manipulação de canvas e evolução progressiva de dificuldade.

🎮 Sobre o Projeto

O Snake PRO é uma versão aprimorada do clássico jogo da cobrinha, contendo recursos modernos como:

Sistema de pontuação

Progressão de fases

Aumento automático de velocidade

Obstáculos dinâmicos

Ranking de jogadores (sessão)

Interface simples e funcional

🚀 Funcionalidades
✔️ Jogabilidade

Controle via teclado (setas direcionais)

Movimento contínuo da cobra

Crescimento ao consumir comida

✔️ Sistema de Pontuação

+10 pontos por comida

Pontuação exibida em tempo real

✔️ Sistema de Fases

A cada 50 pontos:

A fase aumenta

A velocidade do jogo aumenta

Um novo obstáculo é adicionado

✔️ Obstáculos

Gerados dinamicamente a cada fase

Colisão com obstáculo = Game Over

✔️ Ranking

Armazena pontuações durante a sessão

Exibe TOP 5 jogadores

🧠 Tecnologias Utilizadas

HTML5 (Canvas)

CSS3

JavaScript (Vanilla JS)
